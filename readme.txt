A WordPress plugin to make payments with Beam in your online store. 
  
== Description ==
  
This plugin allows payment with Beam in your online store.
Please read the readme file carefully for instructions on how to configure the plugin.
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
Activate, configure with your terminal id and device id and your store is ready to pay with Beam!
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
